# Safient Core SDK

JavaScript SDK to manage and interact with the safes on Safient protocol.

## Installation

```bash
  git clone https://github.com/safient/safient-core.git
  cd safient-core
  npm install
```

## Running Tests

Terminal 1

```bash
  npm run start
```
* Go to http://localhost:3001/initializeDb

Terminal 2

```bash
  npm run test
```
